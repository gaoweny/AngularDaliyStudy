## 相关指令

 - ng-app 定义Angular应用程序
 - ng-model 把元素值绑定到应用程序
 - ng-bind 把应用程序数据绑定到HTML视图
 - ng-init 初始化Angular应用程序变量
 - ng-controller 定义控制器
 - ng-repeat 重复一个html元素
 - ng-show 当为true时，显示html元素
